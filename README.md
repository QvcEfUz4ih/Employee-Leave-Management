# Employee Leave Management System

A Python application that manages and processes 
employee leave requests.

## Features
- Enter employee name and leave details
- Validates leave request (rejects invalid entries)
- Approves leave if sufficient balance available
- Shows remaining leave days after approval
- Notifies when all leave days have been used
- Denies leave with exact shortfall in days shown
- Displays full leave summary

## Technologies Used
- Python 3

## How To Run
1. Download the file
2. Open in Python IDLE or any code editor
3. Run the program
4. Enter employee name, available leave and days requested
5. View leave approval status
